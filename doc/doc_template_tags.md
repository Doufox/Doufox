# 模板标签的使用


基本标签
```
{$site_title}       // 网页标题
{$site_keywords}    // 网页关键字
{$site_description} // 网页描述

{$site_url}        // 网站的网址
{$site_name}       // 网站的名称
{$site_template}   // 网站的主题


```

加载模板标签
```
{template header.html} // 加载当前模板根目录下 header.html 模板文件
{template member/index.html} // 加载当前模板目录下的 member 目录里的 index.html模板文件
```
